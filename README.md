# How to run
1.Clone this repo git clone git@github.com:yangxiaomu/oursToDo.git  
2.cd oursToDo  
3.run npm install  
4.Open oursToDo.xcodeproj in XCode(proposal:The latest version)  
5.Press cmd+r to build it
